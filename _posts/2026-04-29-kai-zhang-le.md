# 开张了

锣鼓喧天，鞭炮齐鸣——好吧也没有，就是安安静静地写了这个博客的第一篇文，然后点了一下发布。

## 这是谁的博客？

我的。小明的。

其实一直想弄个博客，想了大概有……三年？四年？每次都觉得"等我把某某技术学好了再开""等我把页面设计好看了再开""等我找个黄道吉日再开"——然后就没有然后了。

这次不一样。这次我直接动手了。丑就丑着上，内容是王道。再说了，GitHub Pages 默认主题也挺好看的嘛，不丢人。

## 用来干啥？

这个博客大概会有这么几个内容方向：

**技术笔记。** 我是做后端开发的，日常跟各种框架、数据库、消息队列打交道。踩过的坑不记下来，下次还会踩进去同一个坑。这在我身上已经发生了不知道多少次了。俗话说好记性不如烂笔头，写博客就是给自己画一张"这里有坑"的地图。

**项目记录。** 自己折腾了一些小项目，有的是练手，有的是真能用的。把过程写下来，既是整理思路，也是给自己留个纪念。说不定哪天回头看，还能乐一下："当年我写的代码居然长这样？"

**瞎想。** 偶尔也会写点跟代码无关的东西——看了什么书、去了什么地方、想到了什么奇怪的问题。比如：为什么电梯里的关门键好像根本没用？可乐为什么在玻璃瓶里最好喝？这些不重要但又让人好奇的问题，偶尔也会出现在这里。

## 为什么是 GitHub Pages？

因为免费。因为用 Git 管理文章很方便。因为不用操心服务器。因为可以在线编辑。因为支持自定义域名。因为可以绑定自己的域名还带 HTTPS。因为……好吧，说来说去，主要原因还是免费，但免费之外也确实好用。

## 立个 flag

我争取每周至少更新一篇。如果在某个角落的你发现这个博客大半年没动静了，欢迎到 GitHub 上提个 Issue 催更——真的，我需要有人催。我就是那种"没人催就永远在下周"的拖延症患者。

好了，废话不多说。这个博客，从今天起，正式开张。

欢迎光临，随便坐。
